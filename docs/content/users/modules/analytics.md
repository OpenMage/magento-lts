# Analytics modules

## `vianetz/matomo-magento1`
A Magento 1 extension that allows you to get ecommerce transaction data into Matomo analytics.

#### Install [^1]
```bash
composer require vianetz/matomo-magento1
```

[^1]: https://github.com/vianetz/matomo-magento1
